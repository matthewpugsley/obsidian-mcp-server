# Vault Layout Guide

**Summary**
This file provides a guide for structuring an Obsidian Vault to work with the mcp server.

## Folder and File Layout

- __Inbox (collection point)
- _ClaudeWorkflow
  - _index.md (index file for Claude)
  - mcp-state-yyyy-mm-dd (most recent state of the mcp server)
  - Frontmatter Guide.md (guide for how to use frontmatter template)
  - Frontmatter Template.md (Obsidian template)
- Projects (subfolder per project)
- Reference (folders and files for Reference)
- Someday (someday/maybe projects)
- ZZArchive (archive)
